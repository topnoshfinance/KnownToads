# Public Assets

This directory contains public assets for the KnownToads mini app.

## Required Assets

You should add the following image files before deploying:

- `logo.png` - App logo/icon (recommended size: 512x512px)
- `splash.png` - Splash screen image (recommended size: 1080x1920px)
- `og-image.png` - Open Graph image for social sharing (recommended size: 1200x630px)

These should be toad-themed, playful graphics in green colors matching the toadgang brand.

## Favicon

You may also want to add:
- `favicon.ico` - Browser favicon
